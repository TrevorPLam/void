# [MARKETING PRE-FLIGHT: Launch Readiness]

Page: [URL/Route]

Technical Marketing Check:

1. **SEO**: Title ≤60 chars? Meta description ≤160 chars? Canonical tag set?
2. **Performance**: <Image> components used? JavaScript <200KB? Fonts preloaded?
3. **Conversion**: CTA visible above fold? Form labels correct? Analytics events firing?
4. **Social**: OG image 1200x630? Twitter cards configured? Link unfurls correctly?
5. **Mobile**: Touch targets ≥44px? No horizontal scroll? Text readable (≥16px)?
6. **Tracking**: UTM params captured? Conversion pixel firing? A/B test variant tagged?

If any check fails, do not deploy. Fix and re-audit.
